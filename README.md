## 开源协议
ToMart OS遵循GPL 3.0开源协议，这意味着

* 你可以随意复制，使用，修改，分发ToMart OS
* 修改ToMart OS后，如要重新分发，请必须使用GPL 3.0协议开源并标注修改内容
## 最后的话
